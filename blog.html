1. What are the diference between map, filter, forEach, find method?

Ans : The map() method is used to create a new array by applying a function to each element of the original array. The new array has the same length as the original array, and it contains the results of the applied function.

The filter() method is used to create a new array that contains only the elements of the original array that pass a condition.

The forEach() method is used to execute a provided function once for each element of the array. It does not return a new array.

The find() method is used to return the first element in the array that pass a condition. It returns undefined if no such element is found.

2. What are the difference between localStorage, sessionStorage and cookie?

Ans : localStorage and sessionStorage stores data in key-value formate in the user's browser. On the otherhand Cookies are small text files that are stored in a user's browser by a website. localStorage is designed for long term storage and it won't be delete untill we delete in manually. sessionStorage is designed for short term storage and is will be delete if we close the browser. Cookies are designed to store small amounts of data that can be sent back to a website with each request and it has an expiration date.

3. What is event loop in javascript?

Ans : The event loop in JavaScript allows non-blocking I/O operations and asynchronous programming. It maintains a list of functions being executed, and when an asynchronous call completes, the corresponding callback function is added to a message queue. The event loop continuously checks the message queue and executes the corresponding function, resulting in a more efficient application. It's important to be aware of potential issues such as race conditions and use appropriate techniques like Promises and async/await.
